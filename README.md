# Midnight Vault

Crypto Fear & Greed Strategy Dashboard — a real-time portfolio rebalancing tool built with React + Vite.

## Features

- Live Crypto Fear & Greed Index tracking
- Real-time price feeds (BTC, ETH, SOL + custom coins)
- Customizable rebalancing strategies per F&G zone
- Auto-invest with ZK proof simulation
- Wallet connect simulation (Midnight Testnet)

## Tech Stack

- **React 18** — UI framework
- **Vite 6** — Build tool
- **Anthropic Claude API** — Live data via web search

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm run build
```

Framework: **Vite**  
Build Command: `npm run build`  
Output Directory: `dist`
